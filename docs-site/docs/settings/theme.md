# Theme & Appearance

Customize the look and feel of TaskLine.

## Changing the Theme

1. Go to **Settings**
2. Find **Appearance** section
3. Select a theme option

## Available Themes

| Theme | Description |
|-------|-------------|
| Light | Bright, clean appearance |
| Dark | Easier on the eyes in low light |
| System | Follows the system preference |

## Display Options

Additional appearance settings:

- Font size adjustments
- Compact or comfortable view
- Sidebar visibility
