# Account Settings

Manage account details and preferences.

## Changing Username

1. Go to **Settings**
2. Find **Account** section
3. Click **"Change Username"**
4. Enter new username
5. Click **"Save"**

## Changing PIN

1. Go to **Settings**
2. Find **Security** section
3. Click **"Change PIN"**
4. Enter current PIN
5. Enter new PIN (4-8 digits)
6. Confirm new PIN
7. Click **"Save"**

## Delete Account

Permanently delete account and all data:

1. Go to **Settings**
2. Scroll to **Danger Zone**
3. Click **"Delete My Account"**
4. A warning message appears showing this action is permanent
5. Confirm deletion

**Warning:** Deleting an account removes all tasks, categories, and settings. This cannot be undone. Consider exporting data before deleting.
