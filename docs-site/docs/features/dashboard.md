# Dashboard

The Dashboard provides an overview of tasks and productivity statistics.

## Accessing the Dashboard

Click **"Dashboard"** in the sidebar to view the overview.

## Dashboard Features

### Task Summary

- Total tasks count
- Tasks by status (To Do, In Progress, Done)
- Overdue tasks count
- Tasks due today

### Quick Actions

- Add new task
- View all tasks
- Access recent tasks

### Statistics

- Tasks completed this week
- Productivity trends
- Category breakdown

### Recent Activity

- Recently created tasks
- Recently completed tasks
- Upcoming due dates

## Widgets

The Dashboard displays information in organized sections:

| Widget | Description |
|--------|-------------|
| Task Count | Total and status breakdown |
| Due Today | Tasks due today |
| Overdue | Past due tasks |
| Recent | Latest task activity |
