# Categories

Categories help organize tasks with custom labels and colors.

## Creating a Category

1. Go to **Settings**
2. Find the **Categories** section
3. Click **"Add Category"**
4. Enter:
   - **Name** - Category label (example: Work, Personal, Urgent)
   - **Color** - Pick a color for the category
   - **Description** (optional) - Additional details
5. Click **"Save"**

## Editing a Category

1. Go to **Settings**
2. Find the category to edit
3. Click the **Edit** icon
4. Make changes and save

## Deleting a Category

1. Go to **Settings**
2. Find the category to delete
3. Click the **Delete** icon
4. A warning shows how many tasks use this category
5. Confirm deletion

Note: Deleting a category removes it from all tasks but does not delete the tasks.

## Assigning Categories to Tasks

1. Create or edit a task
2. Select a category from the dropdown
3. Save the task

## Viewing by Category

- Tasks display their category label with the assigned color
- Filter tasks by category in List View
- Category colors appear in all views
