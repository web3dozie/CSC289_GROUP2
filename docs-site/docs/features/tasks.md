# Task Management

TaskLine provides comprehensive task management with validation and organization features.

## Creating Tasks

1. Click **"Add Task"** or the **+** button
2. Fill in the task details:
   - **Title** (required) - Name of the task
   - **Description** (optional) - Additional details
   - **Due Date** (optional) - When the task is due
   - **Priority** (optional) - Low, Medium, or High
   - **Category** (optional) - Organize with custom categories
   - **Time Estimate** (optional) - Estimated time to complete

3. Click **"Save"**

## Editing Tasks

1. Click on any task to open it
2. Make changes to any field
3. Click **"Save"** to update

## Deleting Tasks

1. Click on the task
2. Click **"Delete"**
3. Confirm the deletion

## Task Status

Tasks can have three statuses:

| Status | Description |
|--------|-------------|
| To Do | Task not started |
| In Progress | Task being worked on |
| Done | Task completed |

Change status by:
- Clicking the checkbox to mark complete
- Dragging tasks between columns in Board View

## Archiving Tasks

Completed tasks can be archived to keep the workspace clean.

### How to Archive

- Archive completed tasks automatically (enable in Settings)
- Archive manually by clicking **"Archive"** on a completed task

### Viewing Archived Tasks

1. Go to the Archive section
2. View all previously archived tasks
3. Restore tasks if needed by clicking **"Restore"**

### Archive vs Delete

| Action | Result |
|--------|--------|
| Archive | Task is hidden but saved, can be restored |
| Delete | Task is permanently removed |

## Filtering and Search

- Use the search bar to find tasks by title
- Filter by status, priority, or category
- Sort by due date, priority, or creation date

## Data Export and Import

Export tasks in JSON format:

1. Go to **Settings**
2. Click **"Export Data"**
3. Save the JSON file

Import tasks:

1. Go to **Settings**
2. Click **"Import Data"**
3. Select a JSON file
