# Pomodoro Timer

The Pomodoro Timer helps users stay focused with timed work sessions.

## What is Pomodoro?

The Pomodoro Technique is a time management method:

1. Work for 25 minutes (one "pomodoro")
2. Take a 5-minute break
3. After 4 pomodoros, take a longer 15-30 minute break

## Using the Timer

1. Click **"Timer"** in the sidebar
2. Select a task to work on (optional)
3. Click **"Start"** to begin the 25-minute session
4. Focus on the task until the timer ends
5. Take a break when prompted

## Timer Controls

| Button | Action |
|--------|--------|
| Start | Begin the timer |
| Pause | Pause the timer |
| Reset | Reset to 25 minutes |
| Skip | Skip to next session |

## Linking Tasks

Connect the timer to a specific task:

1. Click **"Select Task"**
2. Choose a task from the list
3. Time spent is tracked for that task

## Timer Settings

Customize the timer in Settings:

- Work session length (default: 25 minutes)
- Short break length (default: 5 minutes)
- Long break length (default: 15 minutes)
- Sound notifications
