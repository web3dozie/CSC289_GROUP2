# AI Chat Assistant

TaskLine includes an optional AI chat assistant called "Zedd Mode" for help with task management.

## What is Zedd Mode?

Zedd Mode is an AI-powered assistant that can:

- Answer questions about tasks
- Help organize and prioritize work
- Provide productivity suggestions
- Assist with task planning

## Enabling AI Chat

The AI chat feature requires an AI API URL:

1. Go to **Settings**
2. Find **AI Configuration**
3. Enter an OpenAI-compatible API URL
4. Save settings

Note: This feature is optional. TaskLine works fully without AI enabled.

## Using the Chat

1. Click the **Chat** icon in the sidebar
2. Type a message or question
3. The AI assistant responds with suggestions

## Example Questions

- "What tasks should I focus on today?"
- "Help me prioritize my work tasks"
- "How can I organize my project tasks?"

## Privacy

- AI chat is optional and disabled by default
- No data is sent unless an API URL is configured
- Conversations are stored locally
