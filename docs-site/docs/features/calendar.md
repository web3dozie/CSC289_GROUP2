# Calendar View

The Calendar View displays tasks organized by due dates.

## Accessing Calendar View

1. Click **"Calendar"** in the sidebar
2. Tasks appear on their due dates

## Navigating the Calendar

- Use arrows to move between months
- Click on a date to see tasks due that day
- Today's date is highlighted

## Adding Tasks from Calendar

1. Click on any date
2. A new task form opens with that date pre-selected
3. Fill in task details and save

## Task Display

Tasks on the calendar show:
- Task title
- Priority indicator (color coded)
- Category label

## Views

- **Month View** - See the full month at a glance
- **Week View** - Focus on the current week
- **Day View** - Detailed view of a single day
