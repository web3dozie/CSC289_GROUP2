# Welcome to TaskLine

**Lock in. Get it done. Stay zen.**

TaskLine is a classy, local-first task companion that runs entirely on the local computer, stores data locally, and helps users finish what matters with clean views, smooth interactions, and optional conversational help.

## What is TaskLine?

TaskLine helps users manage daily tasks with features like:

- **Fast and offline** - Local container, no cloud required
- **Private by design** - PIN lock (4-8 digits), configurable auto-lock
- **Task Management** - Create, edit, and organize tasks
- **Multiple Views** - List, Board, Calendar, and Dashboard
- **Pomodoro Timer** - Stay focused with timed work sessions
- **Categories** - Organize tasks with custom categories and colors
- **Optional AI Chat** - Get help from an AI assistant (if configured)
