# Quick Start Guide

Get started with TaskLine in 5 minutes.

## Create an Account

1. Open TaskLine at `http://localhost:3456`
2. Click **"Open App"**
3. On the Welcome page, click **"Sign Up"**
4. Enter a username (example: `testuser`)
5. Enter a PIN code (4-8 digits)
6. Click **"Create Account"**

## Create a Task

1. After logging in, the Dashboard appears
2. Click **"Add Task"** or the **+** button
3. Enter task details:
   - Title (required)
   - Description (optional)
   - Due date (optional)
   - Priority (optional)
   - Category (optional)
4. Click **"Save"**

## View Tasks

TaskLine offers multiple views:

- **Dashboard** - Overview with statistics
- **List View** - All tasks in a list
- **Board View** - Kanban style (To Do → In Progress → Done)
- **Calendar View** - Tasks by due date

## Complete a Task

1. Find the task in any view
2. Click the checkbox or move to "Done" column
3. Optionally add completion notes

## Lock the App

TaskLine automatically locks after a set time. To manually lock:

1. Click the lock icon in the header
2. Enter PIN to unlock
